# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/project-name.git

cd project-name
```

### Create Virtual Environment

```bash
python -m venv venv
```

Windows

```bash

```

Linux/Mac

```bash
source venv/bin/activate
```

